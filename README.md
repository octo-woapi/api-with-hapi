API with Hapi
===

> Example of a Node.js API written with the framework Hapi.


Getting started
---

Install the dependencies
```
$ npm install
```

Run the tests
```
$ npm test
```

Start the project
```
$ npm start # default port is 3000
```

Features
---

Based on the [Katapi](https://github.com/octo-woapi/katapi) kata!
